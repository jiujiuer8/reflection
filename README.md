# 人生复盘系统

一个帮助你每日复盘、追踪学习、探索职业、规划人生的个人成长工具。

## 功能模块

- **📅 今日复盘** - 每日目标、任务清单、完成统计
- **📚 学习追踪** - AI课程、思维心力课程进度记录
- **💼 职业探索** - SWOT分析、职业思考、困惑记录
- **🎯 人生愿景** - Ikigai探索、价值观、人生目标
- **💭 情绪记录** - 情绪日记(CBT)、迷茫时刻记录
- **❤️ 情感探索** - 情感思考、自我认知
- **📊 历史记录** - 数据洞察、趋势图表

## 使用方法

直接在浏览器中打开 `life-review.html` 文件即可使用。

数据保存在浏览器本地存储中，不会丢失。

## 设计理念

基于全网最佳实践调研，整合了：
- OKR目标管理方法
- Ikigai人生意义探索
- SWOT自我分析
- 情绪日记(CBT认知行为疗法)
- 持续迭代优化
